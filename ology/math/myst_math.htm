最神秘的數

　　說到最神秘的數，莫過於：0 , 1 , 「任何實數」 , 「無限」。


< 無限細分 >

任何兩個相異的實數之間，必定存在無限多個實數。


< 0之不可逆 >

0 × 任何實數（包括 0 ） = 0
0 / 任何非 0 實數 = 0
0 / 0 = 任何實數（包括 0 ）

註：第一條式倒轉時，要把「任何實數」分開為「包括 0 」和「不包括 0 」。


< 0、無限，與任何實數 >

0 × 無限 = 任何實數
任何實數 / 無限 = 0
任何正實數 / 0 = 無限大
任何負實數 / 0 = 無限小
